# Laundry Service Website

A simple website for a laundry service.

## What is this?
This project is a small website that shows laundry services, contact information, and service details.

## Pages
- `index.html` - Home page
- `about-contact.html` - About the service and contact info

## Files
- `resources/css/` - styles for the pages
- `resources/js/` - JavaScript files
- `resources/images/` - pictures used on the site
- `vendors/` - external CSS and JS files

## How to use
1. Open `index.html` in your web browser.
2. Click links to see the other pages.
## How to run in VS Code

1. Open this folder in VS Code.
2. Install the Live Server extension if you do not already have it.
3. Open `index.html`.
4. Right-click inside the editor and select **Open with Live Server**.

The page will open in your default browser, and you can see the project live.


## Notes
- The website is simple and easy to use.
- It is made for showing a laundry business online.
